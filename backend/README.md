> 설치한 패키지
```
npm i express
npm i cors
npm i coolkie-parser
npm i nodemon
npm i dotenv
npm i jsonwebtoken
npm i multer
npm i sqlite3
```

> 구분
```
controllers(디렉토리)
group-API.js -> 그룹관련 API
users-API.js -> 인증관련 API
mypage-API.js -> 마이페이지관련 API

models(디렉토리)
group-Queries.js -> 그룹관련 쿼리
users-Queries.js -> 인증관련 쿼리
mypage-Queries.js -> 마이페이지관련 쿼리

index.js(파일)
-> 서버실행 and 라우터
```
